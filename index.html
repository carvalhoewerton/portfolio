Email Sender

In this application, you can send emails using the Java Mail Sender along with Gmail. To activate this feature, you must include a 16-character Gmail API key in the applications configuration. Once configured, you can send emails by providing your email address, the recipients address, the subject, and the message text. After sending an email, the application automatically stores the email details, including sender, recipient, subject, and message text, into a PostgreSQL database. This ensures that a record of each email sent is maintained for future reference or auditing purposes.
